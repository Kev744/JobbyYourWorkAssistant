# Mots d’action pour une lettre de motivation en français

> `→ @Catégorie` = mot déjà présent dans une autre catégorie, référencé pour éviter les doublons.  

## Emploi
Activité, Fonction, Mission, Poste, Tâche, Travail

## Objectif
Ambition, Intérêt, Attrait, Souhait, But, Projet, Envie

## Bénéfice
Avantage, Chiffre d’affaires, Marge, Profit

## Persévérant/e
Accrocheur/se, Combatif/ve, Entreprenant/e, Gagnant/e, Obstiné/e, Tenace, Travailleur/se

## Décider
Arrêter, Choisir, Conclure, Déterminer, Éliminer, Fixer, Juger, Opter, Régler, Résoudre, Trancher

## Vendre
Convaincre, Conclure → @Décider, Négocier, Persuader

## Approche
Cheminement, Direction, Parcours, Progression

## Compétence
Aisance, Faculté, Aptitude, Qualité, Capacité, Connaissance, Compétence

## Entreprise
Affaire, Établissement, Firme, Organisme, Société

## Secteur
Branche, Département, Domaine, Métier, Milieu, Monde, Profession, Spécialité

## Méthode
Appliqué, Logique, Organisé, Rationnel, Rigoureux, Minutieux

## Important/e
Avantageux, Décisif, Essentiel, Nécessaire, Primordial

## Gérer
Acquérir, Amortir, Budgéter, Comptabiliser, Consolider, Économiser, Enrichir, Équilibrer, Exploiter, Gagner, Investir, Optimiser, Rentabiliser

## Organiser
Aménager, Anticiper, Arranger, Coordonner, Distribuer, Établir, Planifier, Préparer, Prévoir, Programmer, Répartir, Structurer

## Superviser
Coordonner → @Organiser, Conseiller, Consulter, Contrôler, Gérer → @Gérer, Surveiller

## Communiquer
Dialoguer, Discuter, Échanger, Écouter, Exprimer, Informer, Interviewer, Négocier → @Vendre, Partager, Rédiger, Renseigner, Transmettre

## Administrer
Classer, Compter, Enregistrer, Établir → @Organiser, Gérer → @Gérer, Inventorier, Ranger, Recenser, Régir†, Répertorier

## Développer
Accroître, Améliorer, Amplifier, Augmenter, Commercialiser, Conquérir, Élargir, Étendre, Déclencher, Décupler, Doubler, Implanter, Lancer, Progresser, Promouvoir, Redresser, Renforcer, Rentabiliser → @Gérer

## Chercher
Analyser, Apprécier, Appréhender, Anticiper → @Organiser, Calculer, Consulter → @Superviser, Définir, Enquêter, Étudier, Examiner, Expérimenter, Juger → @Décider, Observer, Prospecter, Rechercher, Sonder

## Produire
Appliquer, Effectuer, Exécuter, Réaliser

## Conseiller
Clarifier, Comprendre, Diagnostiquer, Éclairer, Écouter → @Communiquer, Guider, Inciter, Orienter, Préconiser, Proposer, Recommander

## Construire
Bâtir, Consolider → @Gérer, Créer, Équilibrer → @Gérer, Élaborer, Réaliser → @Produire, Stabiliser, Structurer → @Organiser

## Former
Animer, Apprendre, Conduire, Développer, Éduquer, Entraîner, Éveiller, Instruire, Sensibiliser, Transformer

## Obtenir
Acquérir → @Gérer, Avoir, Posséder, Réaliser → @Produire, Réussir

## Contrôler
Apprécier → @Chercher, Enquêter → @Chercher, Évaluer, Examiner → @Chercher, Expérimenter → @Chercher, Gouverner, Ordonner, Maîtriser, Mesurer, Prendre en charge, Présider, Prouver, Tester, Valider, Vérifier

## Moderniser
Actualiser, Adapter, Améliorer → @Développer, Réformer, Redéfinir, Réorganiser, Restructurer

## Concevoir
Mettre au point, Mettre en place, Mobiliser, Planifier → @Organiser, Préparer → @Organiser, Programmer → @Organiser

## Négocier
Acheter, Arbitrer, Argumenter, Conclure → @Décider, Consulter → @Superviser, Convaincre → @Vendre, Démontrer, Discuter → @Communiquer, Influencer, Persuader → @Vendre, Placer, Proposer → @Conseiller, Sélectionner

## Diminuer
Baisser, Freiner, Réduire

## Collaborer
Aider, Contribuer, Faciliter, Participer, Seconder

## Diriger
Animer → @Former, Présider → @Contrôler, Agencer, Piloter, Assumer, Manager, Commander, Conduire → @Former, Confier, Coordonner → @Organiser, Définir → @Chercher, Déléguer, Encadrer, Gérer → @Gérer, Gouverner → @Contrôler, Guider → @Conseiller, Impulser, Inspirer

## Créer
Adapter → @Moderniser, Améliorer → @Développer, Concevoir, Construire, Découvrir, Démarrer, Élaborer → @Construire, Fonder, Imaginer, Implanter → @Développer, Initier, Innover, Inventer, Lancer → @Développer, Transformer → @Former, Trouver
